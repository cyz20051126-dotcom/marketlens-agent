# MarketLens AI 来源登记表

本文件说明证据来源结构。完整证据行见 `data/evidence.csv`。

## 来源策略

- 优先使用中文官网、港交所中文公告、中文商业媒体和中文新闻来源。
- 英文来源只在更接近一手公告、中文来源缺失或需要保留原始披露时使用。
- 二手报道、付费墙摘要或推断性风险会降低置信度，必要时标记为 `needs_review`。
